# Utility 模块测试规则文档

> 本文档记录通用业务模块的核心测试规则。
> 生成通用功能相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Utility 模块包含以下核心功能：
- 设置（语言、主题、安全等）
- 通知（推送、消息中心）
- 风控（设备检测、交易拦截）
- 地址簿
- 扫码
- 搜索
- App 升级

## 7. 搜索规则

### 7.1 全局搜索

| 搜索范围 | 规则 |
|---------|------|
| Token | 按名称、符号搜索 |
| DApp | 按名称搜索 |
| 地址 | 按地址搜索 |
| 历史记录 | 搜索历史交易 |

### 7.2 搜索体验

| 规则项 | 规则描述 |
|-------|---------|
| 实时搜索 | 输入即搜索（防抖） |
| 高亮匹配 | 匹配文字高亮 |
| 搜索历史 | 显示最近搜索 |
| 热门推荐 | 显示热门搜索 |

### 7.3 通用搜索框（Wallets / Tokens / dApps / My assets / Perps / Settings / All）

#### 7.3.1 Wallets Tab（账户）

- 地址搜索：
  - 输入非已有账户地址：展示 URL 钱包结果
  - 输入已有账户地址：展示已有账户结果
  - 地址不支持模糊搜索
  - 地址区分大小写
- 账户名搜索：
  - 不区分大小写，支持模糊匹配（例如输入 `acco` 可命中包含 `Account` 的账户）
  - 在单链网络下：账户搜索结果展示地址（前八位…后六位）
  - 在 `All networks` / `BTC` / `LTC` 等多链筛选下：账户搜索结果不展示地址
- 搜索结果点击跳转：
  - URL 钱包结果：跳转至对应网络的 URL 钱包页面
  - 已有账户结果：跳转至 wallet 页面，并切换为当前账户与对应网络

#### 7.3.2 Tokens Tab（代币）

- 输入 Token Symbol/关键词：支持精准与模糊搜索，不区分大小写
- 列表展示字段包含：Token 图标、Symbol、名称、实时币价
- 排序规则：按 Tokens 7天总市值排序
- 支持搜索价格为 `null` 的 Token（如 POWER/AIP）
- 点击搜索结果：跳转到对应 Token 的币种详情 modal

#### 7.3.3 dApps Tab（DApp）

- 域名输入（完整/不完整）：展示访问第三方网站选项与谷歌搜索选项，点击后打开新标签页
- 内容关键词输入（精准/模糊，不区分大小写）：
  - 展示谷歌搜索联想项
  - 展示匹配的 dApp 联想项
- 谷歌搜索项点击：在新标签页打开谷歌搜索，并带入输入内容
- dApp 项点击：在新标签页打开对应 dApp
- 移动端发现 tab 场景：在 `Defi` 页面切换后输入 `uniswap`，点击相关选项应跳转至发现 tab 页面并打开新标签页

#### 7.3.4 My assets Tab（我的资产）

- 搜索范围仅面向当前账户
- 不支持观察地址、私钥、三方钱包作为搜索输入
- 资产搜索：
  - 合约地址：匹配成功
  - Symbol：大小写不敏感
  - 名称：大小写不敏感

#### 7.3.5 Perps Tab（合约）

- 合约 Token 搜索：支持精准与模糊搜索，不区分大小写
- 列表展示字段包含：Token 图标、Symbol、名称、Badge、美元价格
- 排序规则：按服务端返回排序
- 点击结果：跳转至对应 Token 的 Perps 详情页
- 不支持 Perps 的 token（如 USDC/REQ）：不返回搜索结果

#### 7.3.6 Settings Tab（设置）

- 通过钱包关键字搜索：展示钱包和 dapp 账户对齐选项、防护管理选项
- 点击“钱包和 dapp 账户对齐”：跳转至对应设置页面

#### 7.3.7 All Tab（聚合）

- 非 Market 上下文下聚合顺序：`Wallets > Tokens > My assets > dApps`
- Market 上下文下聚合顺序：`Tokens > Wallets > My assets > dApps`
- 展示规则：
  - 无结果板块不展示
  - 每个板块最多展示前 5 个结果
  - 超过 5 个结果展示“显示更多”
  - 点击“显示更多”：切换到对应 Tab，并保留输入内容

## 8. App 升级规则

### 8.1 版本检测

| 规则项 | 规则描述 |
|-------|---------|
| 启动检测 | App 启动时检测新版本 |
| 手动检测 | 设置中手动检测 |
| 更新提示 | 新版本弹窗提示 |

### 8.2 升级类型

| 类型 | 规则 |
|-----|------|
| 强制升级 | 必须升级才能使用 |
| 推荐升级 | 可跳过，后续提醒 |
| 静默升级 | 后台下载，提示安装 |

### 8.3 升级流程

| 步骤 | 规则 |
|-----|------|
| 下载 | 显示下载进度 |
| 安装 | 跳转应用商店/直接安装 |
| 失败处理 | 提示重试 |
