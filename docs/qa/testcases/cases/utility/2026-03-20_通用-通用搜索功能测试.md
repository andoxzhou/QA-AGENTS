# 通用 - 通用搜索功能测试

> 生成时间：2026-03-20
> 规则引用：`docs/qa/rules/utility-rules.md`
> 需求引用：`docs/qa/requirements/Utility-通用搜索功能.md`

## 测试范围说明

**测试覆盖要求**：
- 搜索类型覆盖：地址、账户名、Token Symbol、dApp 域名/内容关键词、资产合约地址/Symbol/名称、Perps 合约 Token、设置关键字
- 覆盖 Tab：`Wallets`、`Tokens`、`dApps`、`My assets`、`Perps`、`Settings`、`All` 聚合
- 覆盖规则：地址无模糊&区分大小写、账户名模糊不区分大小写、Tokens 7天总市值排序、All 聚合顺序&展示数量限制、Perps 支持/不支持 token 行为

---

## 前置条件

- 已存在钱包类型：软件钱包、硬件钱包、隐藏钱包、二维码钱包、导入私钥钱包、观察钱包、外部账户
- 所有钱包均创建账户：`Account #1`、`Account #2`
- 可进行网络筛选：
  - 单链网络模式
  - `All networks` / `BTC` / `LTC` 等多链筛选模式
- `My assets` 当前账户已具备代币资产（用于合约地址/Symbol/名称搜索验证）
- `Perps` 支持的 token 覆盖：比特/eth/黄金/Tsla/et/Go/tc/LA
- `Perps` 不支持 token：`USDC`、`REQ`
- 任意账户可改名为：`abcd1234喊腻&……$!@#^&*(){}?><¥😂ま한국たعَرَبِ`
