# Perps - 投资组合 & 盈亏

> 生成时间：2026-03-26  
> 规则引用：`docs/qa/rules/perps-rules.md`  
> 需求文档：`docs/qa/requirements/Perps-Portfolio&PnL.md`

## 测试范围说明

- **覆盖端**：桌面端 / Web 端 / iOS / Android
- **核心改动**：Perps 原“充值/Deposit”入口逻辑改造为“投资组合与盈亏（Portfolio & PnL）”入口；桌面端为弹窗浮层；移动端为全屏页面；桌面端空账户分支打开充值引导弹窗
- **主要风险**：入口分支（空账户/非空账户）、图表格式化与边距（formatter/margins）多端一致性、统计空态与异常兜底、风险阈值边界（40%/70%）、频繁切换导致旧数据残留
