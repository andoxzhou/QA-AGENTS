# Browser - DApp 浏览器新增翻译设置入口

> 生成时间：2026-03-30
> 更新时间：2026-03-30（桌面端地址栏 / iOS 底栏 / Android 底栏、设置持久化、PrimeFeatures 关联）

## 测试范围说明

**测试覆盖要求**：
- Desktop：地址栏翻译图标点击后 Popover 展示翻译设置并可执行翻译
- iOS：底栏翻译入口出现、点击后面板展示引擎/目标语言/显示模式选项并可触发翻译动作
- Android：底栏翻译按钮支持手势点击，翻译状态切换后按钮图标同步变化（未翻译 / 已翻译）
- 翻译配置持久化：引擎/目标语言/显示模式修改后，重新进入浏览器配置不回退
- PrimeFeatures：DAppTranslate 枚举接入在开关控制场景下入口行为保持一致
