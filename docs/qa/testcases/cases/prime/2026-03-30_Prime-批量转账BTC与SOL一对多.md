## 3. 最小转账金额校验（收款地址输入页）


| 优先级        | 场景                           | 操作步骤                                       | 预期结果                             |
| ---------- | ---------------------------- | ------------------------------------------ | -------------------------------- |
| ❗️❗️P0❗️❗️ | 1. 网络=BTC 或 SOL 2. 进入收款地址输入页 | 1. 粘贴多行“地址+金额” 2. 包含低于最小转账金额的行 3. 点击下一步    | 1. 页面显示最小金额错误提示 2. 停留在当前页        |
| ❗️❗️P0❗️❗️ | 1. 网络=BTC 或 SOL 2. 进入收款地址输入页 | 1. 粘贴多行“地址+金额” 2. 所有金额均 >= 最小转账金额 3. 点击下一步 | 1. 不显示最小金额错误提示 2. 跳转至金额设置/复核流程页面 |
| P1         | 1. 网络=BTC 或 SOL 2. 进入收款地址输入页 | 1. 输入金额=0 2. 输入金额=-1 3. 输入金额=最小值-最小精度      | 1. 每种输入均显示错误提示 2. 当前步骤不可提交       |

## 4. 最小转账金额校验（金额设置页三种模式）


| 优先级        | 场景                          | 操作步骤                                        | 预期结果                     |
| ---------- | --------------------------- | ------------------------------------------- | ------------------------ |
| ❗️❗️P0❗️❗️ | 1. 网络=BTC 或 SOL 2. 已进入金额设置页 | 1. 切换到指定金额模式 2. 输入低于最小转账金额 3. 点击下一步         | 1. 显示最小金额错误提示 2. 下一步不可点击 |
| ❗️❗️P0❗️❗️ | 1. 网络=BTC 或 SOL 2. 已进入金额设置页 | 1. 切换到范围金额模式 2. min 或 max 低于最小转账金额 3. 点击下一步 | 1. 显示最小金额错误提示 2. 下一步不可点击 |
| ❗️❗️P0❗️❗️ | 1. 网络=BTC 或 SOL 2. 已进入金额设置页 | 1. 切换到逐地址编辑模式 2. 任一地址金额低于最小转账金额 3. 点击下一步    | 1. 该地址行显示错误提示 2. 下一步不可点击 |
| P1         | 1. 网络=BTC 或 SOL 2. 已进入金额设置页 | 在三种模式分别输入金额=最小转账金额                          | 三种模式均跳转至下一步页面            |

## 9. 异常与边界
