1. APP 已初始化，已创建钱包账户
2. 切换到 All Networks 网络下
3. 钱包中有 ETH / BTC / SOL 等测试资产

## 支持网络

> 代码来源：`packages/shared/src/utils/bulkSendUtils.ts` — `getBulkSendSupportedNetworkIds()`

**一对多模式**（仅以下网络）：
- EVM：ETH / BSC / Arbitrum / Polygon / Base / Optimism / Avalanche / Linea / zkSync Era
- 非 EVM：Tron / Solana / Bitcoin
- 排除：Lightning Network / Nostr

**多对一 / 多对多模式**：支持除 Lightning Network / Nostr 外的所有网络（代码跳过一对多的网络白名单限制）
