# Market - Token 详情页图表（K 线）

> **模块**：Market（行情）  
> **功能名称**：Token 详情页 K 线图表  
> **版本**：（待填写 App 版本）  
> **测试端**：Android / iOS / Extension（插件端）/ Desktop（桌面端）

---

## 1. 需求背景

用户在 Market 中进入 Token 详情页后，通过 K 线图表查看价格走势、切换周期与指标，并与常用第三方行情源核对数据口径；各端需保证交互流畅与断网恢复能力。

---

## 2. 前置条件

| 条件 | 说明 |
|------|------|
| 账户类型 | 已创建或已导入：助记词钱包、硬件钱包、外部账户、私钥账户、观察账户（按端能力覆盖） |
| 入口 | 进入 Market 模块 → 进入目标 Token 详情页 |
| 网络 | 默认可联网；异常场景单独断网 |

---

## 3. 功能描述

### 3.1 图表数据展示（K 线上方）

| 项 | 说明 |
|----|------|
| 时间区间 | 支持 1m / 15m / 1h / 4h / D / W；需求描述默认选中 1m（**若与「首次加载默认 15m」并存，以产品最终口径为准**） |
| OHLC | 随区间切换变化；O=时段内首个成交价，H=最高价，L=最低价，C=时段内最后成交价；可与 OKX / TradingView 对照 |
| K 线类型 | 默认蜡烛图；可切换其他类型（以产品提供的类型为准） |
| 价格 / 市值 | 支持在价格与市值展示之间切换 |
| 下方时间周期 | 支持 1 日 / 5 日 / 1 月 / 3 月 / 6 月 / 1 年 / All；**时区**：默认展示用户当前时区（**移动端不展示时区相关 UI 时以端上实现为准**） |

### 3.2 基础交互

| 项 | 说明 |
|----|------|
| 加载 | K 线图可加载；需求描述首次默认 15 分钟周期（与 3.1 默认 1m 若冲突需产品定稿） |
| 切换周期 | 切换后图表与数据与所选周期一致 |
| 缩放与平移 | 支持缩放、上下与左右平移查看历史数据 |

### 3.3 技术指标

| 项 | 说明 |
|----|------|
| VOL | 成交量；可与 TradingView / OKX DEX 对照 |
| EMA | 叠加于 K 线，数值计算准确 |
| MACD | DIF / DEA / 柱状图显示正确；可与 TradingView / OKX DEX 对照 |
| RSI | 显示正确；可与 TradingView / OKX DEX 对照 |
| 布林带 | 带宽随波动变化，计算无异常 |
| 多指标组合 | 多指标同时显示无互相破坏 |
| 指标上限 | 达到产品定义的最大指标数量时行为正确（含与第三方对照） |
| 移除指标 | 可移除单个指标且不影响其余指标 |
| 自定义指标 | 全局保存，对所有 Token 详情页生效；切换 Token 后仍保留 |

### 3.4 交互事件

| 项 | 说明 |
|----|------|
| 十字光标 | 各端可移动十字光标；展示时间、O、H、L、C（以端上 UI 为准） |
| 点击 K 线 | 移动端点击蜡烛弹出详情（时间、交易量、涨跌幅等，以产品文案为准） |

### 3.5 数据与异常

| 项 | 说明 |
|----|------|
| 刷新 | K 线数据刷新及时；可与 TradingView 对照 |
| 交易所对照 | 与 Binance / OKX 在 15m / 1h / 24h 等维度对照，误差在文档约定容差内 |
| 时区 | 切换时区后 K 线时间戳与 UTC 对齐无偏差（桌面端等支持时区的端） |
| 大数据量 | 加载约一年 1m 历史数据时图表仍可交互，无明显卡顿 |
| 断网恢复 | 断网后恢复网络，K 线自动补全或重新拉取至完整状态 |

---

## 4. 业务规则

| 规则项 | 描述 |
|--------|------|
| 数据一致性 | 详情页图表与列表 / 详情头部价格在约定口径下一致 |
| 第三方对照 | 对照时需统一标的、网络、周期、时区与 K 线对齐方式 |
| 端差异 | Extension / Desktop / iOS / Android 在系统能力（时区 UI、手势）上允许差异，核心数据口径一致 |

---

## 5. 已知风险

| 风险点 | 说明 |
|--------|------|
| 默认周期口径 | 「上方区间默认 1m」与「首次加载默认 15m」需产品统一，避免用例与实现长期不一致 |
| 数据源差异 | 不同交易所切片时间、未收盘 K 线可能导致 OHLC 细微差异，需约定容差 |
| 硬件钱包 | 仅影响账户上下文，图表一般为只读；若存在权限差异需单独列出 |
