# Swap - 价值下跌提示（Value Drop Warning）需求文档

> 模块：Swap
> 功能：预交易提示弹窗 — 高 Value Drop 风险提示
> 版本：
> 测试端：Web / iOS / Android / Desktop / Extension
> 复现 Token：`0x9b5e262cf9bb04869ab40b19af91d2dc85761722`（NOCK，Base 网络）

---

## 需求背景

Swap 兑换确认前，当检测到高价值下跌（high value drop）时，弹出预交易提示弹窗，让用户明确知晓价值损失风险并通过勾选确认 + 5 秒冷静期倒计时后才可继续交易，降低因流动性不足或价格剧烈波动导致的误操作风险。

---

## 功能描述

### 1. 视觉风险提示
- 触发高 value drop 报警时，弹窗风险提示区域改为**红色 critical 卡片**
- 百分比标题显示**红色**
- Continue 按钮为 **destructive 视觉样式**（红色/警告风格）
- 用户能更直观看到高风险状态

### 2. 勾选确认 + 倒计时机制
- 弹窗中包含确认复选框（如"我知晓风险并继续"）
- 勾选复选框后，Continue 按钮进入 **5 秒倒计时**
- 倒计时期间按钮保持**禁用**状态，按钮文案显示"继续 (N)"（N 为剩余秒数）
- 倒计时结束后按钮变为可点击，用户才可继续交易

### 3. 倒计时重置
- 取消勾选或重复勾选复选框时，倒计时**重置为 5 秒**并重新开始
- 用户每次重新确认都需完成完整等待流程

### 4. 埋点上报
- 点击 Continue：上报 `valueDropTipContinue` 事件
- 点击 Cancel：上报 `valueDropTipCancel` 事件
- 事件包含字段：from/to token 符号、数量、法币值、链 ID、跌幅百分比、勾选状态

---

## 业务规则

| 规则 | 描述 |
|------|------|
| 触发条件 | 当 Swap 报价的 value drop 达到高风险阈值时触发弹窗 |
| 倒计时时长 | 固定 5 秒 |
| 倒计时重置 | 每次勾选/取消勾选都重置为 5 秒 |
| 按钮禁用 | 未勾选或倒计时未结束时 Continue 按钮不可点击 |
| 视觉等级 | critical 红色风格 |
