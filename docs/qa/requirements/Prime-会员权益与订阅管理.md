# Prime - 会员权益与订阅管理

> 模块：Prime  
> App 版本：待定  
> 日期：2026-03-20  
> 测试端：iOS / Android / Desktop / Extension / Web

---

## 1. 需求背景

Prime 是 OneKey 的会员体系，用于向用户提供会员权益展示、会员订阅、续费/取消、订阅状态管理与多端状态同步能力。用户可通过 Prime 页、受限功能引导页或设置入口完成订阅，并在 OneKey ID 体系下查看和管理当前会员状态。

---

## 2. 功能描述

### 2.1 会员权益展示

| 项目 | 说明 |
|-----|------|
| 权益列表 | 展示当前权益展示页已确认的 Prime 会员权益，具体清单见 2.9 节 |
| 对比展示 | 支持普通用户与 Prime 用户权益对比 |
| 权益说明 | 每项权益支持独立说明，具体权益定义见 2.9 节 |
| 权益类型 | 当前已确认权益均为功能解锁类型 |
| OneKey Cloud 展示策略 | OneKey Cloud 功能保留，但不在 Prime 权益展示页展示 |

### 2.2 功能访问控制

| 项目 | 说明 |
|-----|------|
| 权限检查 | 访问 Prime 专属功能时检查订阅状态 |
| 非会员引导 | 非会员访问受限功能时引导进入 Prime 介绍或订阅流程 |
| 试用能力 | 部分功能可支持试用，是否启用以实际产品配置为准 |

### 2.3 订阅入口

| 入口路径 | 说明 |
|---------|------|
| 设置页 | 提供 Prime 会员入口 |
| 权益引导页 | 受限功能触发的 Prime 引导入口 |
| 活动推广页 | 活动场景中的 Prime 订阅入口 |

### 2.4 订阅方式与周期

| 维度 | 说明 |
|-----|------|
| 应用内购买 | 按平台规则唤起 App Store、Google Play 或 webview 支付 |
| 订阅周期 | 支持月付、年付 |
| 价格矩阵 | 不同国家或地区 / 货币的月付与年付价格详见 `docs/qa/requirements/Prime-订阅价格方案.md` |

### 2.5 应用内购买渠道

| 平台 | 说明 |
|-----|------|
| iOS | 唤起 App Store 支付 |
| Android（支持 Google 框架，商店安装） | 唤起 Google Play 支付 |
| Android（支持 Google 框架，APK 安装） | 弹窗选择 Google Play 或 webview 支付 |
| Android（不支持 Google 框架） | 唤起 webview 支付 |
| 其他端 | 唤起 webview 支付 |

### 2.6 订阅状态与管理

| 项目 | 说明 |
|-----|------|
| 状态机 | 未订阅 → 订阅中 → 已订阅 → 到期 → 续费 / 过期；支持取消订阅 |
| 订阅信息 | 展示订阅状态、订阅类型、支付方式 |
| 到期时间 | 取消自动续费后显示；持续订阅中不展示 |
| 取消路径 | 统一通过「管理订阅」进入，不同支付渠道跳转规则不同 |
| 自动续费 | 支持自动续费，需用户授权 |
| 续费提醒 | 到期前提供续费提醒 |
| 手动续费 | 支持手动续费 |
| 取消生效 | 当前周期结束后生效 |
| 权益保留 | 取消后至到期前保留 Prime 权益 |
| 再次订阅 | 支持重新订阅 |

### 2.7 管理订阅与套餐修改

| 项目 | 说明 |
|-----|------|
| 管理订阅入口 | 统一通过「管理订阅」进入 |
| 商店渠道（支持对应商店设备） | 打开对应商店订阅管理 |
| 商店渠道（不支持对应商店设备） | 通过网页打开对应商店订阅管理 |
| webview 渠道 | 发送管理链接邮件到登录邮箱 |
| iOS 月费转年费 | 直接切换 |
| iOS 年费转月费 | 从到期时间开始变更，当期到期后切换 |

### 2.8 支付完成与状态同步

| 项目 | 说明 |
|-----|------|
| 状态刷新 | 支付成功后刷新当前 OneKey ID 的 Prime 状态 |
| 会员资格同步 | 同一 OneKey ID 在多端的 Prime 会员资格与权益状态同步更新，设备限制详见 `docs/qa/requirements/Prime-登录设备限制.md` |
| 状态标识 | 各端 Prime 入口、OneKey ID 入口及相关状态标识保持一致 |
| 多端显示 | Prime 状态标志颜色仅表示当前登录客户端自身状态 |

### 2.9 会员专属功能清单

> 本节定义 Prime 权益展示页展示清单；`OneKey Cloud` 功能继续保留，详见 `docs/qa/requirements/Prime-OneKeyCloud.md`，但不在权益展示页展示。

| 功能 | 说明 | 详细需求 |
|-----|------|---------|
| 批量转账 | 支持 BTC / SOL 一对多批量转账 | `docs/qa/requirements/Prime-批量转账.md` |
| 批量复制地址 | 批量导出软件/硬件/隐藏钱包地址 | `docs/qa/requirements/Prime-批量复制地址.md` |
| 批量撤销授权 | 批量撤销 Token 授权 | `docs/qa/requirements/Prime-批量撤销授权.md` |
| 提高通知上限 | 账户通知上限由 20 提升至 100 | `docs/qa/requirements/Prime-通知上限提高.md` |
| Dapp页面AI翻译 | Dapp 页面内容 AI 翻译能力 | `待补充` |
| Dapp站点风险检查 | 打开 Dapp URL 时执行站点风险检查；当前详细需求沿用风险扫描缓存能力 | `docs/qa/requirements/Prime-dApp站点风险扫描缓存.md` |
| 扩展交易历史（即将推出） | 扩展交易历史展示与查询能力 | `待补充` |
| 导出交易（即将推出） | 支持导出交易记录 | `待补充` |

> 待补充权益：`Dapp页面AI翻译`、`扩展交易历史（即将推出）`、`导出交易（即将推出）`。当前先保留 Prime 权益清单占位，详细需求与专项用例待补充。

### 2.10 关联功能

| 功能 | 说明 | 详细需求 |

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 权益展示 | Prime 页面需展示当前权益展示页已确认的权益列表、对比信息与权益说明 |
| OneKey Cloud 展示策略 | OneKey Cloud 功能保留，但不在 Prime 权益展示页展示 |
| 权限校验 | 访问 Prime 专属功能前需校验会员状态 |
| 入口覆盖 | Prime 需支持设置页、权益引导页、活动推广页等多入口进入 |
| 订阅方式 | 当前已确认支持应用内购买 |
| 应用内购买渠道 | iOS、Android 商店安装 / APK 安装、是否支持 Google 框架及其他端的支付渠道按平台规则区分 |
| 周期方案 | 默认支持月付、年付 |
| 价格方案 | 各国家或地区 / 货币的月付与年付价格以 `docs/qa/requirements/Prime-订阅价格方案.md` 为准 |
| 管理订阅 | 取消订阅统一通过管理订阅入口进入，跳转方式按支付渠道与操作设备能力区分 |
| 续费能力 | 支持自动续费、到期前提醒续费与手动续费 |
| 套餐变更 | iOS 支持月费转年费直接切换、年费转月费到期后生效 |
| 到期时间展示 | 仅在取消自动续费后展示到期时间 |
| 权益保留 | 取消自动续费后，当前周期结束前继续保留 Prime 权益 |
| 状态同步 | 同一 OneKey ID 的会员资格与权益状态需在多端同步更新，设备限制详见 `docs/qa/requirements/Prime-登录设备限制.md` |
| 标志显示 | Prime 状态图标颜色仅表示当前登录客户端自身状态 |
