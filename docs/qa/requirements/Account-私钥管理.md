> 模块：Account

> App 版本：待定  
> 日期：2026-04-14  
> 测试端：Desktop / Extension / iOS / Android / Web

---

## 1. 需求背景

私钥管理覆盖私钥导入、导出、公钥导出以及不同链格式识别，是 Account 模块的安全高风险区域。为保证链类型识别、导入预测、权限验证与导出限制一致，需要形成独立需求。

---

## 2. 功能描述

### 2.1 私钥格式支持

| 链类型 | 格式 | 长度 |
|-------|------|------|
| EVM | `0x` 开头十六进制 | 64 字符（不含 `0x`） |
| BTC | WIF | 51 至 52 字符 |
| Solana | Base58 | 87 至 88 字符 |

### 2.2 私钥导入

| 项目 | 说明 |
|-----|------|
| 格式校验 | 自动识别私钥格式 |
| 链匹配 | 私钥格式必须与目标链匹配 |
| 重复检测 | 导入已存在私钥时定位到对应账户并提示 `该钱包已存在` |
| 导入预测 | 根据输入字符预测匹配网络 |
| 网络补充 | 点击 `找不到想要的网络` 可展开所有网络列表选择 |

### 2.3 私钥导出

| 项目 | 说明 |
|-----|------|
| 权限验证 | 导出前需验证密码或生物识别 |
| 风险提示 | 导出前显示风险警告 |
| 一次性显示 | 退出页面后再次查看需要重新验证 |
| 复制确认 | 用户勾选确认信息前不可确认复制 |

### 2.4 导出细则与公钥导出

| 项目 | 说明 |
|-----|------|
| 网络 / 派生路径 | 可选，默认与钱包首页当前选择一致 |
| 未创建地址 | 未创建地址时私钥栏为空，并提示该账户还没有创建地址 |
| 已创建地址 | 私钥栏默认掩码显示，可切换可见和复制 |
| BTC / fork 链 | 导入私钥的 BTC 及 fork 链账户支持导出对应公钥 |
| 公钥导出入口 | 账户选择器 → 第一个账户三个点 → 查看详情 → 导出公钥 |
| 公钥导出权限 | 私钥 / HD 钱包导出公钥需验证密码；公钥观察账户导出公钥不需要验证密码 |

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 格式优先 | 私钥导入必须先完成格式识别和链匹配 |
| 重复导入处理 | 已存在私钥账户不能重复创建 |
| 导出安全门槛 | 私钥查看、复制和再次查看都需要经过权限验证和风险提示 |
| 未创建地址限制 | 未创建地址时不能展示虚假私钥内容 |
| 公钥导出边界 | BTC / fork 链、公钥观察账户和私钥账户的权限口径按规则执行 |

---

## 4. 已知风险

- 多链私钥格式识别与目标网络预测容易出现误判。
- 未创建地址、掩码显示、复制确认和重新验证之间存在较强状态联动。
- BTC 及 fork 链的公钥导出路径和账户来源容易出现差异。
