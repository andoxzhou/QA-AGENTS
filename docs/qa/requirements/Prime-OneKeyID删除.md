# Prime - OneKey ID 删除

> 模块：Prime  
> App 版本：待定  
> 日期：2026-03-20  
> 测试端：iOS / Android / Desktop / Extension / Web

---

## 1. 需求背景

OneKey ID 删除能力用于支持用户主动注销当前 OneKey ID 账号，并清理与该账号相关的 Prime 身份、云同步状态、邀请码与登录态。删除后，同邮箱可重新注册为新用户，但不继承删除前数据。

---

## 2. 功能描述

### 2.1 入口与前置

| 项目 | 说明 |
|-----|------|
| 入口位置 | 已登录状态下可在 OneKey ID 相关设置中看到删除入口 |
| 未登录态 | 未登录时不显示删除 OneKey ID 入口 |
| 勾选前置 | 需勾选确认项后才可继续删除 |

### 2.2 验证流程

| 项目 | 说明 |
|-----|------|
| APP 密码 | 删除前需先输入 APP 密码 |
| 邮件验证码 | APP 密码校验通过后，向注册邮箱发送验证码 |
| 验证码页 | 显示验证码输入框与 60 秒倒计时 |
| 错误处理 | 错误或过期验证码提示错误 |
| 重发 | 60 秒后可重新发送新验证码 |

### 2.3 删除结果

| 项目 | 说明 |
|-----|------|
| 数据同步 | 其他端数据同步停止 |
| 登录态 | 其他端自动退出 OneKey ID 登录状态 |
| 邀请码 | 原邀请码失效 |
| 已绑定关系 | 已绑定邀请码的钱包保持原绑定关系 |

### 2.4 重新注册

| 项目 | 说明 |
|-----|------|
| 同邮箱注册 | 同邮箱可重新注册 |
| 用户身份 | 重新注册后生成新用户与新邀请码 |
| 数据继承 | 不继承删除前数据与邀请关系 |

---

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 登录前提 | 删除 OneKey ID 必须在已登录状态下执行 |
| 确认门槛 | 未勾选确认项不得进入删除流程 |
| 双重验证 | 删除前需依次完成 APP 密码与邮箱验证码校验 |
| 倒计时 | 验证码页需展示 60 秒倒计时，并在结束后允许重发 |
| 跨端退出 | 删除完成后其他登录端需退出登录 |
| 同步停止 | 删除完成后原账号相关同步行为需终止 |
| 邀请码失效 | 删除账号后原邀请码不再可用 |
| 已绑定保留 | 删除前已绑定邀请码的钱包关系不回滚 |
| 重新注册隔离 | 同邮箱重新注册后视为新用户，不继承旧数据 |

---

## 4. 已知风险

- 删除后其他端退出登录的生效时机可能存在同步延迟
- 邀请码失效范围与已绑定关系保留策略需与推荐系统保持一致
- 删除后重新注册时的旧缓存清理与新用户初始化需重点确认
