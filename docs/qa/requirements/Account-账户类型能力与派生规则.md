# Account - 账户类型能力与派生规则

> 模块：Account  
> 来源：基于 `docs/qa/rules/account-rules.md` 补充整理  
> App 版本：待定  
> 日期：2026-04-14  
> 测试端：Desktop / Extension / iOS / Android / Web

---

## 1. 需求背景

Account 模块同时承载 HD 钱包、硬件钱包、观察账户、外部账户、私钥账户与 Keyless Wallet 等多种账户形态。为保证多端创建、派生、批量创建和自动创建的行为一致，需要对账户能力矩阵、派生标准和账户创建边界进行统一整理。

---

## 2. 功能描述

### 2.1 账户类型与能力矩阵

| 账户类型 | 说明 | 关键能力限制 |
|-----|------|-------------|
| HD 钱包 | 通过助记词派生的钱包 | 支持完整能力 |
| 硬件钱包 | 通过硬件设备管理的钱包 | 需设备确认 |
| 观察账户 | 仅查看余额的账户 | 不支持签名与转账 |
| 外部账户 | 通过第三方钱包连接的账户 | 受连接钱包与网络限制 |
| 私钥账户 | 通过私钥导入的账户 | 受链类型与账户系列限制 |
| Keyless Wallet | 通过社交登录、PIN 与 App Passcode 创建或恢复 | 不支持导出助记词、KeyTag、Lite，不参与云备份与数据传输 |

### 2.2 派生标准支持

| 标准 | 路径格式 | 适用范围 |
|-----|---------|---------|
| BIP44 | `m/44'/coin'/0'/0/index` | 所有链默认路径 |
| BIP49 | `m/49'/coin'/0'/0/index` | BTC P2SH-SegWit |
| BIP84 | `m/84'/coin'/0'/0/index` | BTC Native SegWit |
| BIP86 | `m/86'/coin'/0'/0/index` | BTC Taproot |

### 2.3 派生一致性与批量创建

| 项目 | 说明 |
|-----|------|
| 同助记词一致性 | 同一助记词在不同设备和平台派生的地址必须一致 |
| 多账户派生 | 支持按 index 递增创建多个账户 |
| 链间隔离 | 不同链的派生路径彼此独立 |
| 同助记词账户关联 | 同助记词导入的软件 / 硬件 / QR 钱包中，对应账户头像保持一致 |
| 地址预览模式 | 每页显示 10 个地址和余额，支持翻页 |
| 高级模式 | 支持直接设置生成数量、起始账户与派生路径 |
| 数值边界 | 页码、起始账户、生成数量都按规则自动修正到允许范围 |

### 2.4 自动创建与路径切换

| 场景 | 说明 |
|-----|------|
| All networks 添加账户 | 默认创建 BTC 四种派生路径、EVM-BIP44、Solana-BIP44、Tron 地址 |
| 指定网络添加账户 | 在默认地址基础上补充当前选中网络地址 |
| 聚合网络 | BTC / LTC 自动创建所有派生路径地址 |
| 多派生网络 | EVM / Solana / Kaspa 自动创建当前派生路径地址 |
| 系列网络 | Cosmos / DOT 自动创建对应系列地址 |
| 网络限制 | 当前钱包或设备不支持所选网络时，默认地址仍可创建，并提示当前网络地址创建失败 |
| DOT 限制 | 15 / 18 / 21 位助记词软件钱包不支持创建 DOT 地址，仅 12 / 24 位支持 |

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 账户能力边界 | 各账户类型的查看余额、转账、签名、导出私钥和导出助记词能力按能力矩阵执行 |
| 二维码钱包归类 | QR Wallet / QR Hidden Wallet 按硬件钱包连接形态归入硬件钱包规则 |
| 派生稳定性 | 同助记词、同路径、同平台或跨平台派生结果需保持一致 |
| 批量创建边界 | 页码、起始账户和生成数量超过范围时必须自动修正到允许值 |
| 自动创建规则 | All networks、聚合网络、多派生网络与系列网络的地址创建需按对应规则补齐 |
| 受限网络处理 | 不支持网络时不能阻断默认地址创建 |

---

## 4. 已知风险

- 多链派生与自动创建逻辑组合后，容易出现地址数量、路径和首页展示不一致。
- 页码、起始账户和生成数量的边界值较大，需要重点验证自动修正和剩余可创建范围。
- DOT 助记词位数限制、聚合网络和系列网络的自动创建规则容易出现端间差异。
