## 1. 需求背景

软件钱包与助记词能力是 Account 模块的核心基础，覆盖创建、导入、未备份状态、安全限制和手动备份。为保证助记词输入、校验、备份和未备份钱包限制行为一致，需要形成独立需求口径。

---

## 2. 功能描述

### 2.1 软件钱包创建

| 项目 | 说明 |
|-----|------|
| 助记词生成 | 新建软件钱包默认生成 12 词助记词 |
| 创建完成 | 创建完成后直接进入钱包首页 |
| 备份标记 | 新建软件钱包默认标记为未备份钱包 |

### 2.2 助记词导入

| 项目 | 说明 |
|-----|------|
| 手动输入 | 输入字符后联想 BIP39 单词，点击联想自动跳到下一格 |
| 粘贴导入 | 助记词按空格自动分发到各输入框，粘贴后清空剪贴板 |
| clear / 切换位数 | 清空当前输入；切换位数时同时清空当前输入 |
| 实际校验 | 以实际输入助记词为准，不仅依赖位数选择器 |
| 错误提示 | 非词库、非英文字符、无效单词、词数错误、校验和错误都显示 `无效助记词` |
| 重复导入 | 已存在相同钱包时定位到对应钱包并提示重复 |
| TON 助记词 | 24 位 TON 助记词导入为 TON 单链私钥钱包，不创建 HD 软件钱包 |

### 2.3 安全规则

| 项目 | 说明 |
|-----|------|
| 截屏限制 | 助记词页面禁止截屏 |
| 剪贴板清理 | 助记词粘贴后自动清空剪贴板 |
| 暗码显示 | 非焦点输入框统一显示 4 位暗码 |
| 二次验证 | 查看、备份、导出类助记词流程都需要密码或生物识别验证 |
| 风险确认 | 复制或导出前展示风险确认 |

### 2.4 未备份钱包

| 项目 | 说明 |
|-----|------|
| 首页展示 | 不展示账户余额，显示备份提示、绑定邀请码、教程 / 客服入口 |
| 基础能力 | 切换 tab、创建账户、创建地址不受影响 |
| 受限能力 | 复制地址、理财认购、导出公钥 / 私钥、DApp 签名等操作触发备份提示 |
| 列表限制 | 未备份钱包不出现在发送代币 / NFT 的我的账户列表 |
| 状态变化 | 完成备份后移除未备份标记；数据传输不会移除该标记 |

### 2.5 手动备份

| 项目 | 说明 |
|-----|------|
| 入口 | 设置、账户选择器、首页备份提示都可进入 |
| 验证 | 进入前先验证 App 密码或生物识别 |
| 展示 | 顶部显示钱包或账户名称，助记词明文展示并带序号 |
| 复制 | 复制前展示风险确认，复制后剪贴板内容与助记词一致 |
| 校验 | 未备份新钱包查看助记词后需进入随机词校验；已备份或导入钱包再次查看不强制校验 |

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 新建默认状态 | 新建软件钱包后直接进入首页，并默认处于未备份状态 |
| 导入优先级 | 助记词导入以实际输入内容为准 |
| 助记词错误口径 | 非法词库、词数错误和校验和错误统一提示 `无效助记词` |
| TON 特例 | TON 助记词导入不创建 HD 软件钱包 |
| 安全优先 | 助记词查看、复制和导出前必须经过安全验证和风险确认 |
| 未备份限制 | 未备份钱包的受限能力和发送列表限制需严格生效 |
| 手动备份闭环 | 查看助记词、风险确认、随机词校验和返回再进入行为需完整闭环 |

---

## 4. 已知风险

- 粘贴导入、位数切换、联想输入组合后，容易出现输入框残值与实际校验不一致。
- 未备份钱包首页状态与发送列表、导出能力、DApp 签名限制之间存在联动风险。
- 助记词复制、风险确认与随机词校验的串联流程较长，容易出现返回态异常。
