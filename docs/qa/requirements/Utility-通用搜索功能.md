# Utility - 通用搜索功能需求文档

> **模块**：Utility（通用）
> **功能名称**：通用搜索（Wallets / Tokens / dApps / My assets / Perps / Settings / All）
> **版本**：待定
> **测试端**：全端（iOS / Android / Extension / Desktop / Web）

---

## 1. 需求背景

通用搜索入口覆盖多个业务域（账户、代币、DApp、我的资产、合约、设置项），并在 `All` 汇总页按模块场景提供跨域聚合排序与展示数量限制。为保证用户在不同网络/不同模块上下文输入关键字时，搜索结果类型识别、展示字段、排序规则与跳转逻辑一致，需要固化通用搜索的业务规则。

---

## 2. 前置条件

- App 下已存在以下钱包类型：软件钱包、硬件钱包、隐藏钱包、二维码钱包、导入私钥钱包、观察钱包、外部账户
- 所有钱包均已创建账户 `Account #1`、`Account #2`
- 账户网络筛选：
  - 可切换到单链网络
  - 可切换到 `All networks` 以及 `BTC`、`LTC` 等链筛选
- `My assets` 测试资产准备：
  - 当前账户已具备至少以下链上代币资产（若当前账户无资产，可先进行自定义添加）：
    - `eth / USDT / 0xdac17f958d2ee523a2206206994597c13d831ec7`
    - `eth / USDC / 0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48`
    - `eth / ETHEREUM / ethereum`
    - `btc / bitcoin / bitcoin`
    - `nervos / TEST / off`（合同地址按产品准备）
    - `sol / PYUSD / paypal`（合同地址按产品准备）
- `Perps` 测试资产准备：
  - 服务端支持的合约 Token 覆盖比特/eth/黄金/Tsla/et/Go/tc/LA 等搜索命中
  - 不支持的 token 包含 `USDC`、`REQ`
- 可对任意账户执行“改名”为包含特殊字符的名称：`abcd1234喊腻&……$!@#^&*(){}?><¥😂ま한국たعَرَبِ`

---

## 3. 功能范围

通用搜索入口包含以下 Tab：

- `Wallets Tab`：搜索账户（地址/账户名）
- `Tokens Tab`：搜索 Token（Symbol/名称的精准与模糊）
- `dApps Tab`：搜索 DApp（域名/内容关键词）
- `My assets Tab`：搜索当前账户的我的资产（合约地址/Symbol/名称/网络）
- `Perps Tab`：搜索 Perps 支持的合约 Token
- `Settings Tab`：搜索与钱包相关的设置项
- `All`：跨 Tab 汇总展示（在非 Market 与 Market 场景下的聚合顺序不同）

---

## 4. 业务规则

### 4.1 Wallets Tab - 地址与账户名搜索

#### 4.1.1 输入非已有账户地址

- 在 `Wallets Tab` 输入一个“未在现有账户中创建”的地址：
  - 展示 `URL 钱包` 结果
  - 展示字段包含：账户图标、网络图标角标、网络名、地址（前八位…后六位）
- 点击该搜索结果后：
  - 跳转到对应网络的 URL 钱包页面

#### 4.1.2 输入已有账户地址

- 在 `Wallets Tab` 输入“属于已有账户”的地址：
  - 展示已有账户结果
  - 展示字段包含：账户图标、钱包图标角标、钱包名/账户名、资产价值、地址（前八位…后六位）
- 点击该搜索结果后：
  - 跳转到 wallet 页面
  - 并切换为当前账户以及对应网络

#### 4.1.3 地址搜索边界

- 地址不支持模糊搜索：
  - 输入不完整地址（例如删除尾部字符）应返回无结果
- 地址区分大小写：
  - 输入与已有地址仅在大小写上不同的值应返回无结果

#### 4.1.4 账户名搜索（单链 vs 多链）

- 在单链网络下输入账户名（如 `Account #1`）：
  - 展示所有钱包下包含 `Account #1` 的账户
  - 字段包含：账户图标、钱包名/账户名、资产价值、地址（前八位…后六位）
- 在 `All networks` / `BTC` / `LTC` 下输入账户名（如 `Account #1`）：
  - 展示所有钱包下包含 `Account #1` 的账户
  - 不展示地址字段，仅展示：账户图标、钱包名/账户名、资产价值

#### 4.1.5 账户名模糊搜索与特殊字符

- 支持模糊搜索且不区分大小写：
  - 输入 `acco` 可命中包含 `Account` 的账户结果
- 任意账户改名为包含特殊字符的名称后：
  - 在通用搜索输入可命中的关键字后，应能够检索到该账户

---

### 4.2 Tokens Tab - Token 搜索

- 在 `Tokens Tab` 输入以下关键字：`USDC/USDT/btc/eth/Doge/et/us/th/dc/dt`
  - 展示所有精准搜索与模糊搜索相关的 Token
  - 搜索结果不区分大小写
  - 展示字段包含：Tokens 图标、Symbol、名称、实时币价
  - 列表按 `Tokens 7天总市值` 排序
- 点击 Token 搜索结果：
  - 跳转到对应 Token 的币种详情 modal
- 输入价格为 `null` 的 Token（如 `POWER/AIP`）：
  - 支持搜索并展示结果

---

### 4.3 dApps Tab - 域名与内容关键词搜索

- 输入完整域名（如 `https://www.baidu.com`）：
  - 展示访问第三方网站选项、谷歌搜索选项
  - 点击后打开新标签页
- 输入不完整域名（如 `https://uniswap.`）：
  - 展示访问第三方网站选项、谷歌搜索选项
  - 点击后打开新标签页
- 输入完整内容搜索（如 `uniswap/Jupiter/Venus`）：
  - 展示谷歌搜索与精准/模糊联想的 dApp
  - 不区分大小写
- 谷歌搜索项：
  - 谷歌图标 + 文案 `Search "输入内容"`
  - 点击后在新标签页打开谷歌搜索输入内容
- dApp 搜索项：
  - dApp 图标 + dApp 名称
  - 点击后在新标签页打开对应 dApp
- 输入不完整内容关键词：
  - `jup` 可模糊命中包含 `jup` 的 dApp
  - `uni` 可模糊命中包含 `uni` 的联想 dApp
  - `j` 可模糊命中包含 `j` 的联想 dApp
  - `swap` 可模糊命中包含 `swap` 连续单词的 dApp
- 移动端场景：
  - 在“发现 tab”切换到 `Defi` 页面后，在通用搜索输入 `uniswap`
  - 应展示访问第三方网站、谷歌搜索选项
  - 点击后跳转到发现 tab 页面，并在新标签页打开

---

### 4.4 My assets Tab - 当前账户资产搜索

- 搜索范围仅面向当前账户
- 不支持以下搜索来源：
  - 观察地址
  - 私钥
  - 三方钱包
- 在单链/所有网络下搜索当前账户：
  - 没有资产时可先自定义添加后再进行验证
- 代币资产搜索规则：
  - 代币合约地址：搜索成功
  - Symbol 大小写：搜索成功，不区分大小写
  - 代币名称大小写：搜索成功，不区分大小写

---

### 4.5 Perps Tab - 合约 Token 搜索

- 在 `Perps` 合约 tab 输入以下关键字：`比特/eth/黄金/Tsla/et/Go/tc/LA`
  - 展示所有精准与模糊搜索相关 Token
  - 不区分大小写
  - 分行展示字段包含：Tokens 图标、Symbol、名称、Badge、美元价格
  - 按服务端返回排序
- 点击 Perps Token 搜索结果：
  - 跳转到对应 Token 的 Perps 详情页
- 输入不支持 Perps 的 token（如 `USDC/REQ`）：
  - 不显示搜索结果

---

### 4.6 Settings Tab - 设置项搜索

- 在 `Settings Tab` 输入钱包相关关键字：
  - 展示钱包和 dapp 账户对齐选项
  - 展示防护管理选项
- 点击“钱包和 dapp 账户对齐”：
  - 跳转到对应的设置页面

---

### 4.7 All Tab - 跨 Tab 聚合展示

#### 4.7.1 非 Market 模块上下文

- 在非 Market 模块打开时，检查 `All` tab 搜索结果
- 输入 `Account` / `ETH`
  - 聚合顺序为：`Wallets > Tokens > My assets > dApps`
  - 没有搜索结果的板块不展示
  - 最多显示前 5 个结果
  - 当某板块超过 5 个结果时展示“显示更多”按钮
- 点击“显示更多”按钮：
  - 切换到对应 Tab
  - 保留输入内容

#### 4.7.2 Market 模块上下文

- 在 Market 模块打开时，检查 `All` tab 搜索结果
- 输入 `Account` / `ETH`
  - 聚合顺序为：`Tokens > Wallets > My assets > dApps`
  - 展示规则同非 Market 上下文（无结果不展示、最多前 5、显示更多与保留输入）

---

## 5. 边界与异常规则（本需求范围内）

- 地址不支持模糊搜索
- 地址区分大小写
- Tokens 支持搜索价格为 `null` 的 Token
- Perps 对不支持的 token 不返回搜索结果
- All Tab 展示遵循聚合顺序与展示数量限制
